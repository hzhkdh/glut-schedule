# 体测评分标准表密度优化设计

## 目标

在不改变数据解析、横纵向滚动、固定列和固定表头行为的前提下，减少表一、表二左侧固定列及权重说明占用的视觉空间。

## 设计

- “等级”和“分值”继续作为两个独立列显示，不合并单元格。
- “等级”列宽由 `68dp` 调整为 `52dp`。
- “分值”列宽由 `64dp` 调整为 `48dp`。
- 固定区总宽由 `132dp` 调整为 `100dp`。
- 权重说明字号由 `12sp` 调整为 `11sp`，行高由 `18sp` 调整为 `16sp`。
- 保留现有颜色、边框、内容、位置和滚动行为。

## 验收标准

- “优秀”“良好”“及格”“不及格”均能完整显示。
- “等级”和“分值”仍有清晰的独立单元格边界。
- 横向滚动时两列继续固定在左侧，纵向滚动与惯性不受影响。
- 权重说明更紧凑且仍可读。
- Debug 构建通过；本次不生成 Release。

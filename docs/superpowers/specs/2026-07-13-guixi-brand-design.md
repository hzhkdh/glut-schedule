# “桂系一站式”品牌替换设计

## 目标

移除 App 中作为品牌使用的学校名称、简称和校徽，建立原创的“桂系一站式”视觉标识，同时保持原应用包名、升级路径和功能不变。

## 品牌资源

- 桌面图标使用已选定的“桂花与网格”B 方案。
- 菜单顶部使用同一图形的透明简化版，仅保留深蓝四瓣网格与中央金色桂花。
- 不使用校徽、学校缩写、建校年份、校名标准字、校园建筑或其他学校专属视觉元素。

## 文案

- App 名称及菜单标题改为“桂系一站式”。
- 菜单副标题保留“简单 高效 纯粹”。
- 关于页改为独立非官方工具说明。
- FAQ 中作为品牌出现的“桂工课表”“桂工同学”改为中性表述。
- 业务解释中必要的教务系统来源描述保持事实性，不作为品牌标识使用。

## 技术范围

- 更新 Android legacy 与 adaptive launcher icon 资源。
- 更新菜单顶部图片资源与相关 Compose 文案。
- `applicationId` 与版本号保持不变，避免破坏覆盖升级和本地数据。
- 不改变任何业务、网络、解析或存储逻辑。

## 验收标准

- 桌面和菜单内不再展示旧校徽或“桂工课表”品牌名。
- 菜单简化图标在 38dp 下仍清晰。
- Release 构建、签名与版本检查通过。

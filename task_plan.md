# 周次课表解析健壮性调查与修复

## 目标

定位并修复周次课表的漏课、跨周污染和特殊教室解析问题，确认个人课表是否仍有必要，以及桂林/南宁春秋作息时间的权威来源；保持版本号 0.20.0。

## 成功标准

- 2026·秋第1周与教务周次课表逐项一致，不出现不属于第1周的英语课，周一3-4节机器视觉存在。
- 2024·秋第6周周三1-2节“大学生安全教育1”能够显示，特殊中文场地不导致丢课。
- 课程有效性不依赖教室必须是数字或非空，并有自动化回归测试。
- 明确个人课表剩余职责以及各学期作息时间的权威数据来源。
- 桂林、南宁均使用同一套经过验证的规则。
